# 🍽️ Smart Canteen Pre-Ordering System

An Android-based **Smart Canteen Pre-Ordering System** developed using **Kotlin**, **Jetpack Compose**, and **Firebase** to streamline campus food ordering. The application enables students and faculty to pre-order food, make UPI payments through external payment apps, track order status in real time, and collect orders efficiently using QR codes and Order IDs.

Developed as a **Minor Project** at **B V Raju Institute of Technology (BVRIT)**.

---

## ✨ Features

### 👨‍🎓 Student Module
- OTP-based Login using Firebase Authentication
- Browse digital menu by category
- Search and filter menu items
- View food details with pricing
- Add items to cart
- Customize food orders (special instructions)
- Place food orders in advance
- UPI payment through external payment apps
- Real-time order tracking
- QR Code & Order ID based order collection
- Order history
- Profile management
- Feedback submission

---

### 👨‍🍳 Staff Module
- View incoming orders
- Update order status
- Manage active orders
- View completed orders
- Process QR-based order collection

---

### 👨‍💼 Admin Module
- Secure Admin Login
- Add new food items
- Update menu items
- Delete menu items
- Upload food images
- Manage inventory availability
- View customer feedback

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/aashrithnapa/BVRIT-Smart-Canteen-System.git
```

2. Open the project in Android Studio.

3. Configure Firebase by adding your own `google-services.json`.

4. Sync Gradle.

5. Build and Run the application.
